# CITS4404-Assignment2

## Setup

---

1. Create & activate the virtual environment

   ```
   virtualenv venv
   source venv/bin/activate
   ```

2. Install the required packages

   ```
   pip install -r requirements.txt
   ```

3. Run any of the bots
   ```
   python3 bot.py
   python3 bot_gabe.py
   python3 btc_daivik.py
   ```
