# CITS4404-Assignment2

## Setup
1. Clone this project
```
git clone https://github.com/ivy0305/CITS4404-Assignment2.git
```
2. Switch to kc_bot branch
```
git checkout kc_bot
```
3. Install the required packages
```
pipenv install
```
4. Activate the virtual environment 
```
pipenv shell
```
5. Run the bot
```
python run_test.py
```
